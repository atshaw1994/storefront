# Storefront

A demo storefront application built to showcase coding abilities using Ruby on Rails.

## Features

- Product listing
- Shopping cart
- User authentication
- Order management

## Requirements

- **Ruby**: 3.2.2
- **Rails**: 7.1.3
- **Node.js**: 18.x
- **Yarn**: 1.22.x
- **PostgreSQL**: 14.x

## Setup

1. Clone the repository:

   ```sh
   git clone https://github.com/yourusername/storefront.git
   cd storefront
   ```

2. Install dependencies:

   ```sh
   bundle install
   yarn install
   ```

3. Set up the database:

   ```sh
   rails db:setup
   ```

4. Start the server:
   ```sh
   rails server
   ```

## Running Tests

```sh
bundle exec rspec
```
